Welcome to DAMPP

Dockerized Apache, MariaDB, PHP & PHPMyAdmin

What is DAMPP?

DAMPP is a solution using Docker to host a complete local development environment including:

  • Apache (web server)
  • MariaDB (MySQL database)
  • PHP
  • PHPMyAdmin (database management interface)

Access the services

  • Local website
  • PHPMyAdmin
  • Dashboard
  • MariaDB: port 3306 (host DAMPP_db)

Where to put your web files?

Place your files in the www folder of your DAMPP project. They will automatically be accessible from http://localhost.

Default credentials

  • MariaDB: user root, password root
  • PHPMyAdmin: connects automatically using the MariaDB credentials.

Documentation

Used versions:

  • Apache: 2.4.62
  • MariaDB: 10.4.32
  • PHP: 8.2.12
  • PHPMyAdmin: 5.2.1
